À propos de Nikto

Nikto est un scanner de vulnérabilités pour serveurs web open source qui effectue des tests complets contre les serveurs web pour de multiples éléments utilisé par votre module Python.

Fonctionnalités disponibles:
  • Plus de 6700 éléments/fichiers potentiellement dangereux
  • Vérification des versions obsolètes de serveurs
  • Détection de problèmes de configuration
  • Tests de vulnérabilités communes
  • Scan de plugins et modules
  • Tests SSL/TLS spécialisés
Module Nikto Intégré
Scan Basique
Utilise run_nikto_scan(url, "basic")
Commande: nikto -h [URL] -C all
Scan Rapide
Utilise nikto_quick_scan(url)
Commande: nikto -h [URL] -T 1
Scan Complet
Utilise nikto_full_scan(url)
Commande: nikto -h [URL] -plugins @@ALL
Scan SSL
Utilise nikto_ssl_scan(url)
Commande: nikto -h [URL] -ssl
